# FlexFrameAnimation

Android 平台，帧动画库，实时渲染，自由业务控制

## 特性

- 🚀 **高性能**：低资源占用，实时渲染技术，首帧即可实时显示
- 📦 **自定义格式**：采用 .ffab（FlexFrameAnimation Bundle）格式的自定义资源格式
- 🎮 **自定义播放器**：内置灵活的动画播放器，支持多种控制模式

## 典型业务场景

- UI 设计输入是图片序列帧，而非矢量动画资源
- 序列帧图片尺寸大，数量多，使用 Android 原生动画框架占用资源多，OOM 且丢帧，不满足业务要求
- 实时响应白天黑夜模式切换，播放动效无缝切换
- 多层帧动画叠加，实现复杂的动画效果
- 需要对动画进行自由业务控制，如裁剪、缩放、旋转等
- 入场动画，随进程启动实时渲染

## 目录结构

`ffa-bundle` ffab 帧动画资源文件，存储一个序列帧的图片资源，要求：

- 图片序列帧尺寸相同
- 图片序列帧数量不超过 65535 张
